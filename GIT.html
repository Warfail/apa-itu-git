Apa Itu Git

Pada dasarnya kita bisa menganggap GIT itu sebagai sebuah repository atau pengimpan secara cloud tentang projek atau tugas yang kita kerjakan. secara umum GIT itu memiliki fungsi sebagai sebuah tempat dimana kita bisa melacak mencatat setiap perubahan file di code atau sourcode kita, yang tersimpan secara cloud seperti di gitlab ataupun di github.

Memang tidak salah, tapi begini ketika kita mulai mengerjakan sebuah projek dengan skala yang besar maka kita akan membutuhkan kerja sama tim. Karena balik lagi kita hanya manusia biasa, bukan robot. Setiap manusia ada batasnya sendiri oleh karena itu GIT dapat memudkan kita untuk berkolaborasi dan bekerja sama dalam menyelesaikan suatu projek yang besar. Karena semua orang dapat mengaksesnya secara realtime dan bisa dari manapun dan kapanpun, sehingga jika ada perubahan mendadak terhadap sebuah web, aplikasi, atau apapun itu. Kita dapat langsung memperbaikinya ataupun mengeceknya secara realtime.

Beberapa Command Git yang Perlu diketahui

mungkin ini agak panjang tapi ini wajib diketahui untuk command GIT dapat kita eksekusi secara GUI atau CLI. pada bagian ini saya akan membahas sedikit soal CLI.

Git Command

  • Git init
    • git init pada dasaranya digunakan untuk menginisialisasi repository hal ini penting karena kita ingin memberitahu kepada repo kita bahwa kita ingin mengirim sebuah file
  • Git Status
    • ini juga penting agar kita tidak kehilangan jejak tentang apa yang kita buat terhadap projek kita
  • Git add .
    • command ini kita gunakan ketika kita sudah selesai mengedit projek kita atau jika kita ingin upload file ke repo kita
  • Git Commit
    • untuk simpelnya git commit ini akan memberikan beberapa log tentang perubahan yang kita lakukan terhadap projek kita
  • Git push
    • untuk mengirim file ke repo kita, tapi sebelum kita bisa langsung push kita harus "git add ." kemudian "git commit" baru kita bisa lanjut ke "git push"
  • Git clone
    • git clone berfungsi untuk menyalin file dari cloud atau dari repo kita secara global.
  • git pull
    • git pull semacam command pada git bash untuk menyinkronisasikan file yang ada di server atau repo kita agar sama dengan projek yang kita buat
