Apa Itu Git

Pada dasarnya kita bisa menganggap GIT itu sebagai sebuah repository atau pengimpan secara cloud tentang projek atau tugas yang kita kerjakan. secara umum GIT itu memiliki fungsi sebagai sebuah tempat dimana kita bisa melacak mencatat setiap perubahan file di code atau sourcode kita, yang tersimpan secara cloud seperti di gitlab ataupun di github.

Mengapa Kita Perlu Belajar GIT

oke jadi mengapa kita perlu belajar GIT? kan bisa kita simpan di komputer atau laptop kita sendiri tanpa perlu seribet itu? Memang tidak salah, tapi begini ketika kita mulai mengerjakan sebuah projek dengan skala yang besar maka kita akan membutuhkan kerja sama tim. Karena balik lagi kita hanya manusia biasa, bukan robot. Setiap manusia ada batasnya sendiri oleh karena itu GIT dapat memudkan kita untuk berkolaborasi dan bekerja sama dalam menyelesaikan suatu projek yang besar. Karena semua orang dapat mengaksesnya secara realtime dan bisa dari manapun dan kapanpun, sehingga jika ada perubahan mendadak terhadap sebuah web, aplikasi, atau apapun itu. Kita dapat langsung memperbaikinya ataupun mengeceknya secara realtime.
